# Лабораторная работа №2

**Состав лабораторной:**
- src
    - Main.java - точка входа в программу. Создаёт битву и добавляет по три покемона в команды союзников и противников, затем запускает битву
    - Pokemons
        - Nidoran.java - класс покемона Nidoran
        - Nidorino.java - класс покемона Nidorino
        - Nidoking.java - класс покемона Nidoking
        - Onix.java - класс покемона Onix
        - Steelix.java - класс покемона Steelix
        - Xurkitree.java - класс покемона Xurkitree
    - Moves
        - DoubleTeam.java - класс атаки Double Team
        - DragonBreath.java - класс атаки Dragon Breath
        - IceBeam.java - класс атаки Ice Beam
        - IceFang.java - класс атаки Ice Fang
        - Leer.java - класс атаки Leer
        - Lunge.java - класс атаки Lunge
        - QuiverDance.java - класс атаки Quiver Dance
        - RockThrow.java - класс атаки Rock Throw
        - SludgeBomb.java - класс атаки Sludge Bomb
        - Swagger.java - класс атаки Swagger
- Pokemon.jar - базовые классы и симулятор сражения (Документация в формате [javadoc](https://se.ifmo.ru/~tony/doc/$0) для ознакомления)
- Docs - документация моего проекта в формате javadoc
