# Лабораторная работа №2

**Состав лабораторной:**
- src
    - Main.java - главный класс приложения, содержащий точку входа в программу
    - Act.java - класс, управляющий основным сценарием симуляции
    - Buns.java - неизменяемая запись (Record), представляющая объект "Плюшка"
    - BunsNotFreshException.java - исключение, указывающее на попытку поедания несвежей плюшки
    - CoffeeRanAwayException.java - исключение, сигнализирующее о неудаче в процессе приготовления кофе
    - EmotionalState.java - интерфейс, определяющий управление эмоциональным состоянием объекта
    - Food.java - интерфейс, определяющий взаимодействие персонажа с плюшками и кофе
    - FrekenBok.java - класс, представляющий персонажа "Фрекен Бок"
    - Karlsson.java - класс, представляющий персонажа "Карлсон"
    - Kid.java - класс, представляющий персонажа "Малыш"
    - Kitchen.java - класс, представляющий локацию кухни
    - LivingRoom.java - класс, описывающий конкретную локацию — комнату Малыша
    - Location.java - абстрактный базовый класс, представляющий локацию (место действия) в симуляции
    - LocationOvercrowdedException.java - исключение, указывающее на попытку превышения максимальной вместимости локации
    - Locations.java - перечисление доступных локаций (мест действия) в симуляции
    - Mood.java - перечисление возможных эмоциональных состояний (настроений) персонажей
    - MovingTo.java - интерфейс, определяющий способность объекта перемещаться в пространстве
    - Persona.java - абстрактный класс Persona, представляющий базовую сущность персонажа в симуляции
- javadoc - документация моего проекта в формате javadoc
